## Texas Building Code Updates 2026: The Baseline Agents Need

The phrase "Texas building code updates 2026" is already circulating through MLS Facebook groups and broker meetings, but most agents still lack clarity on what actually changed, who enforces it, and what their personal liability exposure looks like. Texas does not mandate a uniform statewide residential building code the way states like California or Virginia do. Instead, the state operates under a patchwork system where municipalities adopt and amend the International Residential Code (IRC) and International Building Code (IBC) editions at the local level, authorized by Chapter 214 of the Texas Local Government Code.

What happened in 2026 is that dozens of Texas cities — including Dallas, Houston, San Antonio, Austin, Fort Worth, and El Paso — formally adopted portions of the 2024 IRC/IBC editions with local amendments. These adoptions affect energy efficiency standards, wind resistance ratings in coastal zones, electrical panel requirements, and egress window specifications. For agents, the practical impact lands squarely on disclosures, inspection objections, and contract negotiations.

## Why This Matters More Than Agents Think

**Building code changes do not retroactively apply to existing homes, but they change the conversation around inspections and repairs.** When a buyer's inspector flags an issue that now conflicts with the 2026-adopted code, the buyer's agent faces a choice: negotiate repairs to current code, accept the condition as-is, or walk. The distinction between "code at time of construction" and "current code" is one that agents routinely botch in repair amendment negotiations.

Under TREC rules, specifically Sections 531.2 and 535.156 of the Texas Administrative Code, licensees have an affirmative duty to disclose material facts. An agent who knows a property's electrical system was permitted under a now-superseded code and has documented safety issues cannot stay silent. The Texas Real Estate Commission (TREC) has disciplined agents for exactly this kind of omission — the penalty ranges from a formal reprimand to license revocation under Section 1101.652 of the Texas Occupations Code.

This is not hypothetical risk. TREC's enforcement database shows a steady increase in complaints tied to failure to disclose property condition issues, and code-related defects are a growing subset of those filings.

## Key 2026 Code Changes by Category

Not every code revision matters equally to a real estate transaction. The changes most likely to surface during inspections or appraisals fall into a few categories. Here is a comparison of the most transaction-relevant updates adopted by major Texas metros in 2026 versus the prior code editions they replaced:

| Category | Prior Standard (2018 IRC) | 2026 Adopted Standard (2024 IRC) | Transaction Impact |
|----------|--------------------------|----------------------------------|-------------------|
| Energy Efficiency | R-30 ceiling insulation minimum | R-38 ceiling insulation in climate zones 2-3 | Appraisal adjustments, buyer negotiation leverage |
| Electrical | 200-amp service standard for new builds | 200-amp minimum with EV-ready circuit requirement | Inspection flags on homes without EV infrastructure |
| Egress Windows | 5.7 sq ft minimum opening | 5.0 sq ft minimum (reduced), but sill height max lowered to 44 inches | Bedroom compliance reclassification |

| Smoke/CO Alarms | Interconnected alarms required in new builds | Sealed lithium battery alarms required (10-year life) | Seller repair requests on older alarm systems |

Agents working new construction in [growing suburban markets](/blog/where-deals-break-down) need to verify that builders have pulled permits under the 2026 adopted editions, because a certificate of occupancy issued under the wrong code version can create title and insurance problems at closing.

## How This Hits TREC Forms

The TREC Seller's Disclosure Notice — Form OP-H — remains the primary disclosure vehicle. Sellers answer questions about structural components, roof condition, electrical systems, and environmental hazards. Nothing in the 2026 code updates changes the form itself, but the answers sellers provide must reflect reality under the current code environment.

**Section 4 of Form OP-H asks about walls, ceilings, floors, and foundation — all areas where insulation and structural requirements shifted.** If a seller recently renovated and pulled permits, those permits were issued under the 2026 code. Any work that failed final inspection or was done without permits entirely must be disclosed. Agents who help sellers complete OP-H need to ask pointed questions about recent work.

The TREC One to Four Family Residential Contract (Form 20-17) includes Paragraph 7, the property condition section, which references the seller's disclosure and establishes the buyer's right to inspections. When repair negotiations arise, agents draft amendments using TREC's Amendment form (Form 39-9). The language in these amendments must specify whether repairs will be brought to "current code" or "code at time of original construction." This single phrase can swing a repair bill by thousands of dollars.

For [agents managing multiple active deals](/blog/track-multiple-active-deals), tracking which properties fall under which local code edition is a documentation burden that compounds quickly.

## Common Mistakes Agents Make With Building Code Issues

Five errors show up repeatedly in TREC enforcement actions and E&O claims related to building codes.

**First, agents confuse "grandfathered" with "no disclosure required."** A home built in 1998 is grandfathered under 1998 codes, but if the seller knows the roof sheathing fails to meet even that era's wind standards, the defect still must be disclosed. Grandfathering protects a homeowner from being forced to upgrade to current code — it does not eliminate the duty to disclose known problems.

Second, agents advise sellers to make repairs without permits. When a seller fixes a cited electrical issue by hiring an unlicensed handyman, the agent who suggested that route has created a liability chain. Under Section 1101.652(b)(2) of the Texas Occupations Code, advising a client to take an action that violates local ordinances — including permit requirements — constitutes grounds for disciplinary action.

Third, listing agents market room counts that include bedrooms that no longer meet egress requirements under the locally adopted 2026 code. A room with a window that met the old 5.7 square foot requirement but sits at a 48-inch sill height (above the new 44-inch maximum) is no longer code-compliant as a bedroom if the home is being re-evaluated after a renovation permit.

Fourth, buyer's agents submit repair amendment requests citing "current building code" without specifying which jurisdiction's code applies. In unincorporated areas of Texas counties, the IRC may not apply at all — some counties have no building code enforcement. Writing an amendment that demands "code compliance" in an area with no adopted code is unenforceable and creates contract ambiguity that can [blow a closing](/blog/true-cost-of-a-blown-closing).

Fifth, agents fail to verify the certificate of occupancy date on new construction, assuming the builder's marketing materials are accurate. When a builder started a project under the 2018 IRC but did not receive a CO until after the city's 2026 adoption date, disputes arise over which standards apply. The answer depends on the permit date, not the completion date — but agents who do not check create problems.

## What Brokers Need to Audit and Enforce

Brokers carry supervisory liability under Section 1101.803 of the Texas Occupations Code. If an agent in their office repeatedly fails to handle code-related disclosures correctly, the broker faces discipline alongside the agent. This is not abstract — TREC has suspended broker licenses for systemic compliance failures.

**Brokers should audit three specific areas immediately.** First, review how agents in your office complete or assist with TREC Form OP-H. Pull a random sample of 10 recent listings and check whether Section 4 and Section 7 answers align with permit records available through the relevant municipality's online portal. Houston, Dallas, Austin, and San Antonio all maintain searchable permit databases.

Second, audit repair amendments (Form 39-9) for language that creates ambiguity about code standards. If your agents are writing "bring to code" without specifying which code edition or jurisdiction, you have a systemic training gap. Create a standard operating procedure that requires agents to identify the applicable code edition and cite the municipality.

Third, verify that your office's new construction transaction files include copies of the certificate of occupancy showing the applicable code edition. For [agents you are training on timelines and compliance](/blog/training-new-agents-timelines), building code verification should be a checklist item on every new-build closing file.

| Audit Item | What to Check | Red Flag |
|------------|---------------|----------|
| Seller's Disclosure (OP-H) | Sections 4 and 7 accuracy vs. permit records | "No" answers on properties with open or failed permits |
| Repair Amendments (39-9) | Specificity of code references | Generic "bring to code" language without jurisdiction cited |
| New Construction Files | CO date vs. permit date vs. code adoption date | CO issued after 2026 adoption with permits pulled under prior code |
| MLS Listings | Bedroom counts vs. egress compliance | Rooms marketed as bedrooms that lack compliant egress |
| Agent Training Records | Documentation of code-related CE or office training | No training completed in the 12 months surrounding code adoption |

TREC requires brokers to maintain transaction records for four years under Section 535.2. If an enforcement complaint arrives in year three, you need those files intact and auditable.

## Liability, Deal Cancellation, and Fines

The consequences of mishandling building code issues in a Texas transaction range from annoying to career-ending. At the low end, a buyer discovers a code issue during the option period and terminates under Paragraph 23 of the TREC contract — the seller keeps the option fee but loses the deal and market time. At the high end, an agent faces a TREC complaint, an E&O claim, and potential civil litigation.

TREC fines for violations of the License Act can reach $5,000 per violation under Section 1101.702 of the Texas Occupations Code. License suspension or revocation sits on the table for repeat offenders or egregious failures. The [Texas Real Estate Commission's enforcement page](https://www.trec.texas.gov/enforcement) publishes disciplinary actions, and code-related disclosure failures appear regularly.

**E&O insurance does not always cover intentional non-disclosure.** Most policies exclude claims where the agent knowingly concealed a material defect. If an agent knew about a code violation, advised the seller not to disclose it, and the buyer later sues, the agent may be personally liable for damages with no insurance backstop.

Deal cancellation is the most common outcome. Buyers who discover code issues during due diligence often terminate rather than negotiate. The seller loses time, the listing agent loses commission, and the buyer's agent starts over. Agents who understand building code implications before listing — and price or disclose accordingly — close faster and with fewer fallout risks.
